# Feature Recommendations (Post-1.0.0)

## High Value / Low Risk

1. Complete migration of major dialogs to stylesheet-driven theming.
2. Add focused integration tests for drag/drop edge cases.
3. Consolidate repeated CSS blocks in built-in themes.

## Medium-Term

1. Additional plugin API test harness for compatibility checks.
2. Performance profiling for very large vault trees and note lists.

## Not in Scope (for now)

- Cloud sync or collaborative editing.
- Multi-user conflict resolution.
- Heavy framework re-architecture.
